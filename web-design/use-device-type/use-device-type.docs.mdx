## A React Hook to fetch the current device type

Returns the current device type.

### Component usage

In this example calling the hook once returns a string that tells us what the current device type is. 

```js
import { useDeviceType } from './use-device-type';

const device = useDeviceType();

<h1>{device}</h1>
```
